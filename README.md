# API de Notas

API REST simple para gestionar notas personales.

## Tecnologías

- Node.js
- Express
- JSON (almacenamiento en archivo)

## Instalación

```bash
npm install
npm start
```

## Endpoints

- GET /api/notas - Listar todas las notas
- POST /api/notas - Crear nueva nota
- GET /api/notas/:id - Obtener nota por ID
- PUT /api/notas/:id - Actualizar nota
- DELETE /api/notas/:id - Eliminar nota
